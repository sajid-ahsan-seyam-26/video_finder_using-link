# video_finder_using-link
📺 YouTube Launcher (Tkinter)
A simple, lightweight Python GUI application that allows users to paste a YouTube URL and instantly open it in their default web browser. Built with Tkinter for the interface and the Webbrowser module for functionality.

🚀 Features
Simple UI: Clean and minimal interface.

Instant Play: Launches videos directly in your preferred browser (Chrome, Firefox, Edge, etc.).

Error Handling: Alerts the user if the input field is empty.

Zero External Dependencies: Uses standard Python libraries—no pip install required!

🛠️ Requirements
Python 3.x

Tkinter (Usually comes pre-installed with Python)

📥 How to Run
Clone or Copy: Save the code into a file named video+finder.py.

Run the script:

Paste & Play: Copy a YouTube link into the text box and hit the Play Video button.
